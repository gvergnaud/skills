# gvergnaud skills

Reusable AI agent skills by Gabriel Vergnaud.

## Install

```bash
npx skills@latest add gvergnaud/skills
```

The repository is private for now. The install command works for accounts with
access to the repository and will work for everyone once the repository is made
public.

## Skills

- [typescript-code-quality](./skills/typescript-code-quality/SKILL.md) - Guidance for writing and refactoring TypeScript with maintainability, type safety, explicit error handling, local reasoning, and strong module boundaries.
